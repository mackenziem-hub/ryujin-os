Every visual asset to build the booking and production calendars to match the approved mockups: a themeable SVG icon library plus Higgsfield illustrations, on the Ryujin cockpit aesthetic.

23 SVG icons (24x24, stroke currentColor so accent color is driven by CSS) + 4 illustrations. Full spec of all 66 catalogued assets (incl. reused CSS pieces) is in docs/calendar-asset-manifest.md.
