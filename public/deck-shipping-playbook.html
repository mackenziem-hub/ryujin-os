Play 1 · the canonical loop

Plan mode first. Argue the plan, then one-shot it.

Boris Cherny's verbatim loop: stay in plan mode (read-only), go back and forth until the plan is right, then switch to auto-accept edits and let Claude one-shot the PR. The plan is where you catch right-prompt-wrong-intent before 800 lines exist in the wrong direction.

Route planning to Opus, execution to Sonnet (opus-plan). Mandatory near auth, payments, data access, or any migration.

Play 2 · prove it with tests

Write the failing test first. Each red-to-green is unambiguous feedback.

Claude defaults to implementation-first, so you must prompt it: write failing tests for the behavior, confirm they fail, commit, then implement until green without touching the tests. That loop is what makes overnight autonomous runs safe, because the model has a ground-truth signal to iterate against.

TDD Guard (a PreToolUse hook) blocks any implementation edit when no failing test exists. This would have caught the May 11 agent-chat template-literal break.

Play 4 · spend discipline

Route by job, not by habit. The 90/10 rule.

Sonnet · 90% of work
default
Opus · the hard 10%
50+ file refactor
Haiku · discovery
15x cheaper

Sonnet handles standard generation, single-file bugs, tests, CRUD. Reserve Opus for novel architecture, giant refactors, and regulatory-critical output. Use Haiku for file, grep, and glob subagents, where it is indistinguishable from Opus and far cheaper.

Worktrees fan out parallel work, but cap at 3 to 5 before the context-switching tax eats the gain. Forked subagents share the parent cache, so children 2-N cost roughly 10% of normal.
Play 5 · protect the cache

Most wasted spend is cache you nuked yourself. Guard the prefix.

01

Do not edit CLAUDE.md mid-session. It sits in the cached prefix, so every edit nukes the cache for the rest of the conversation. Today's plan belongs in a session note.

02

Compact at seams, not under duress. Run /compact between finished sub-tasks while the cache is warm. The TTL is 5 minutes now, so walking away to think means a cold write.

03

Batch independent tool calls into one turn. Reads, searches, and git ops should ship together. Separate turns add round-trips for nothing.

04

Keep CLAUDE.md lean and audit MCP. Past ~150 lines, rules drop silently. Unused MCP servers reload up to 18K tokens of schema every single turn.
