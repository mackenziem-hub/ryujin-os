Every price runs through the engine, and Mac says go before it ships.

Make it read like Mac

The voice rules that make it read like Mac wrote it.

01

Positive framing only. Say the benefit ("the price is fixed"), never the negative ("no surprises").

02

King regards sign-off. A why-now stated positively. The third-generation roofer line where it fits.

03

No em dashes. Periods, commas, parentheses, and hyphens carry the sentence instead.

After it ships

A sent proposal is frozen. To change it you clone, you never edit.

The customer must always see exactly what was sent. To change a photo, a colour, or a price line: clone to a new draft, edit the draft, send the new link.

The pattern
Clone → edit the draft → send the new link.
The first link stays frozen and exactly as the customer first saw it.
