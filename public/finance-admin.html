Daily Overhead

Used by the P&L view to amortize fixed costs across the period. Currently tenant_settings.daily_overhead (already exists; default $90/day).

Payment Terms

Default deposit % applied on contract send. Net days = how many days after install the final balance is due (0 = due on completion).

Supplier Credit Limits

Soft caps used by Cashflow agent to flag when a supplier balance is approaching credit limit. Edit per-supplier on the Production · Materials side once those are catalogued.

P&L Assumptions

Used when paysheets aren't yet linked to the estimate. Real P&L uses paysheet totals when available.
